draw, meme, quote

======================

download:

simple:
node editor pull

pull all of type:
node editor cat=draw

pull from ID to ID:
node editor startid=5 cat=quote endid=7

=======================

upload:

node editor push draw

node editor push draw t="Have fun together" a=loffyllama l="www.google.com"

(image link from clipboard)
node editor push meme wi

(video link from clipboard)
node editor push meme wv

=====param:
nd: to not delete file after push
sa=aslam
wi="www.image.jpg"
wv="www.video.mp4"
ol: onlyOverrideLatestMedia
